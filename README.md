# books simple demo app

# Getting Started
Make sure you have [Nodejs](https://nodejs.org/)

```
$ git clone https://github.com/dennis-b/booksApp.git # or clone your own fork
$ cd booksApp
$ npm install
$ gulp build
$ run on local server # you may use gulp webserver task
```
